# ❤️ Heart Disease Risk Assessment AI Agent

## Project Description
This project predicts the risk of heart disease using a Machine Learning model and provides an easy-to-use Streamlit web interface.

## Features
- Predicts heart disease risk
- User-friendly Streamlit interface
- Machine Learning-based assessment
- Real-time predictions

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- NumPy
- Pandas

## Files
- app.py : Main Streamlit application
- heart_disease_rf_model.pkl : Trained Random Forest model
- scaler.pkl : Feature scaler
- requirements.txt : Required Python packages

## How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
